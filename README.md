# Basic React Projects

This repository contains a collection of **small, focused React projects** built using **Vite, React, and Tailwind CSS**.  
The purpose of this repo is to strengthen core React concepts, modern tooling usage, and UI development skills through hands-on projects.

Each project is maintained in its **own folder** with a dedicated setup and README.
